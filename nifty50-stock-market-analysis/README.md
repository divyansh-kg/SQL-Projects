# NIFTY 50 Stock Market Analysis Using SQL

## Project Overview
This project analyzes 5 years of historical stock market data for selected NIFTY 50 companies using SQL. The analysis focuses on price behavior, 
return patterns, trading activity, volatility, and risk-adjusted performance metrics.

## Dataset
- Historical daily stock market data
- Time Period: May 2021 – April 2026
- Companies Analyzed:
  - TCS
  - Reliance
  - SBI
  - Sun Pharma
  - L&T

## Tools Used
- Microsoft SQL Server (MSSQL)
- SQL

## Key Areas of Analysis
### 1. Data Exploration
- Dataset validation
- Duplicate checks
- Historical coverage analysis
- Record distribution analysis

### 2. Price & Performance Analysis
- Average closing prices
- Highest & lowest closing prices
- Average daily returns
- Largest single-day gains & losses
- Closing price range analysis

### 3. Trading Activity Analysis
- Average trading volume
- Peak trading activity
- Volume consistency analysis
- Daily return volatility analysis

### 4. Advanced Comparative Insights
- Risk-adjusted return analysis

## SQL Concepts Used
- SELECT
- WHERE
- GROUP BY
- ORDER BY
- Aggregate Functions
- Statistical Functions
- Data Type Conversion
- Filtering & Sorting
